# Frontend

This frontend is a React + Vite business dashboard for the Bank Marketing Subscription Intelligence Platform.

## Responsibilities

- present executive portfolio KPIs
- surface high-conversion customer segments
- support individual customer scoring
- support scenario planning and explainability
- provide batch scoring review with pagination

## Local Development

```bash
npm install
npm run dev
```

Default local URL:

- `http://127.0.0.1:5173`

## Environment Variables

- `VITE_API_BASE_URL` for the backend API base URL
- `VITE_MLFLOW_UI_URL` for the MLflow UI URL
